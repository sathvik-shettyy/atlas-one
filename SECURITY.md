# Security Policy

## Supported Versions

Security fixes are provided for the latest `main` branch.

## Reporting a Vulnerability

- Do not disclose vulnerabilities publicly.
- Open a private security advisory or contact maintainers directly.
- Include reproduction steps, impact, and suggested remediation if known.

## Security Baseline

- Zero Trust gateway controls all app access.
- MFA and RBAC enforced by policy engine.
- CI includes static analysis and dependency scanning.
